First, the buttons that will open the tabs. To create the container of these buttons: <ul class="tabs [POSITION-HERE]"></ul>. There are 3 positions for the buttons: Left, Right and Center. Change [POSITION-HERE] to the position you want.

Now, the buttons. To create the first button: <li class="first current"><a href="#tabsampleleft1">Tab 1</a></li>. In the class attribute, there are two values: first, for the first button and current for the button where the linked tab is active. Generally only the first button is the current button when a page loads, so use this as it is.

For all other buttons (except the last), you can use with no classes: <li class=""><a href="#tabsampleleft2">Tab 2</a></li>.And in the last button, use the last class: <li class="last"><a href="#tabsampleleft4">Tab 4</a></li>

Note: #tabsampleleft1, #tabsampleleft2 and #tabsampleleft4 reffers to the tabs each button will open. You can name it as you want and with numerations (or any other way you like), but be sure that these referrals are the same of the ID of the tabs they will open.

And for last, the tabs: <tab id="tabsampleleft1" class="tab-content clearfix" style="display: block;">I'm the tab 1!</tab>. Use the ID like the links used in buttons that will open each tab.

Only the first/active tab can have the display: block;. For the others use: display: none;.

Done! You have a working tab menu.

Fixed

Use onclick="$(`#ELEMENT-ID`).toggle();return false" in any element you want to call the menu when clicked.

Use id="ELEMENT-ID" class="navmenu fixed" in your menu. The ID needs to be the same of the used in the onclick event, as seen in ELEMENT-ID. The fixed class makes your menu fixed.

Note: Please remove the style="display:block" from your menu, or it will appear when the page loads, instead of when is clicked.
